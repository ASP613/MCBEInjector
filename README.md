## MCBEInjector

This is just a "weekend-project" type of project, obviously it could do with much refinement but it works at the present time on Minecraft Bedrock 26.31.

The goal is to be efficient, and I am satisfied that this project is reasonably so; the binary should run on most Windows installations, does not require the C Runtime Library, and is just over 5kb in size. 
At runtime, memory usage is fairly minimal, though one may find that the program does not look especially fancy.
